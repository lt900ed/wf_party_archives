緊急事態宣言解除されたから、大人数で崩壊域に遊びに行こうぜ！！

というわけで、キクノのマルボ出るたびフィーバーゲージ増加アビのおかげでぽんぽんフィーバー入れる安定パーティです。
かばうマルボがたくさんあるので、回復量少なくても大丈夫。
メイミーやライア、スイゼンのアビと灼鉞によってマルボのHPマシマシ、キクノの4アビで全属性耐性までついてきます。
灼鉞3凸でもいけました。

回復役は正直誰でもいいと思います。
リャオ、ノエネ、アニバアルク、ジン、ハリシャ試して全員クリア済みです。
ハリシャが一番クリアタイム早かったです。多分ウェイトが一番軽くてマルボ出せる数が多いから？

マナボ2はクリア時
スイゼン - - -
メイミー 1 0 -
キクノ完凸
ハリシャ 5 - -
ライア 1 3 0
ハナビ 0 0 0
でした。必須級なのは恐らくキクノの4アビ。他に取るならマルボHP上昇系がいいと思います。

ハナビの採用理由はウェイト激軽でマルボ5個出せるからです。リリルを採用して結晶をとっとと潰すのもありだと思います。マルボは出ませんがこちらもウェイトめちゃ軽でゲージ加速も僅かながらあるので使える子です。
